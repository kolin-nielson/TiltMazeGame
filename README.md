## Getting Started

### Prerequisites

- Node.js (v14 or later)
- npm or yarn
- Expo CLI
- Expo Go app on your mobile device (or emulator/simulator)

### Running on a Device

1. Install the Expo Go app on your mobile device
2. Scan the QR code displayed in the terminal
3. The app will open in Expo Go
